# BookStore

**Live Demo:** Click [here](https://bookstore-frontend-al1q.onrender.com/)

A full-stack bookstore e-commerce web application built with modern web technologies, featuring responsive design, seamless frontend-backend integration, user authentication, cart management, order handling, and an admin dashboard.

## Tech Stack
- React.js, React router
- Tailwind CSS
- Node + Express.js
- MongoDB Atlas
- Motion

## Features
- Responsive and modern UI
- Product listing with filtering and sorting
- Add to cart functionality
- User authentication (Login/Register)
- User profile management:
  - Current order with invoice,
  - Cancel/return order functionality
- Checkout system with:
  - Delivery form
  - Coupon code support
  - Invoice generation & PDF invoice download

- Admin Dashboard
  - Admin authentication
  - Dashboard analytics and statistics
  - Manage and view books data, orders and coupons
